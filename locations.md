# Config Locations Reference

A complete mapping of every backed-up config to its macOS source location.

## Shell

| Config | macOS Location | Dotfiles Path |
|--------|---------------|---------------|
| Zsh config | `~/.zshrc` | `zsh/.zshrc` |
| Zsh plugins | `~/.zsh_plugins.txt` | `zsh/.zsh_plugins.txt` |
| Bash profile | `~/.bash_profile` | `.bash_profile` |
| Bashrc | `~/.bashrc` | `.bashrc` |

## Intentionally Not Tracked

These common `~/.config/` directories exist but are deliberately excluded:

| Directory | Reason |
|-----------|--------|
| `~/.config/configstore/` | Auto-generated by npm |
| `~/.config/iterm2/` | Using Ghostty instead |
| `~/.config/op/` | 1Password CLI (contains secrets) |
| `~/.config/git/` | Using `~/.gitconfig` instead |
